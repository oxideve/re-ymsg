
# ReYMSG

Re-YMSG is a lightweight Node.js-based server emulator designed to restore core communication functionality for the classic Yahoo! Messenger 5.5 client.


![Logo](https://i.ibb.co/RFzx3Dp/Immagine-2026-05-28-232018.png)


## Currently supported versions

#### Updated on: 28/05/2026


| Version   | Release date    |                 |
| :-------- | :------- | :------------------------- |
| 4.1 and below  | 08/2002 | ❌ - Currently **not** supported |
| 5.5 | 08/2002 | ✅ - Supported |
| 5.6 - 5.9 | 08/2002 | 🟡 - Not tested|
| 6.0 and above | 08/2002 | ❌ - Currently **not** supported|


> I'm currently developing Re-YMSG for **Yahoo Messenger 5.5.1244**: it means that this *could* work on previous versions, but NOT on newer versions. Check the above table.
